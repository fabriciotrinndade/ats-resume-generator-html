## 什么是 ATS Flow？

ATS Flow 是一个轻量级、开源的简历构建器，专为希望完全控制简历的开发者和技术人员设计，无需依赖付费构建器或限制性编辑器。

使用在线视觉编辑器构建简历，或编辑结构化 JSON 以获得完全可重现性。

## 系统生成：

- 兼容 ATS 的简洁 HTML
- 可点击链接的打印就绪 PDF
- 用于可移植性的结构化 JSON

无需订阅。无需跟踪。无锁定。

## 🚀 入门

```bash
# 克隆仓库
git clone https://github.com/fabriciotrinndade/ats-resume-generator-html.git
cd ats-resume-generator-html

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 📖 使用方法

1. 访问 http://localhost:3000
2. 使用视觉编辑器创建简历
3. 导出为 HTML 或 PDF
